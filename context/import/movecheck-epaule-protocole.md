# MoveCheck — Protocole Bilan Épaule

> Généré le 13 juin 2026, basé sur revues systématiques et méta-analyses récentes.
> À utiliser comme référence clinique pour le module épaule de MoveCheck.

---

## Partie 1 — Tests sentinelles (diagnostic par pathologie)

### Cluster 1 : Impingement / Coiffe antérieure

**Hawkins-Kennedy**
- Se 58-79% / Sp 59-66% / DOR 2.86 (meilleur test impingement)
- Patient : coude à 90°, rotation interne passive du bras
- Faisabilité distance : bonne (aide d'un proche ou auto-appliqué)

**Painful arc (arc douloureux)**
- Se 53-96% / Sp 34-77%
- Abduction active, patient marque le secteur douloureux (60-120°)
- Faisabilité distance : excellente (visible par caméra)

**Empty Can / Jobe**
- Se 69-79% / Sp 66-76%
- Résistance auto-appliquée (autre main) ou objet lesté
- Faisabilité distance : bonne

**Drop Arm Sign**
- Se 27% / Sp 88-98% / LR+ 3.3
- Sentinel de rupture complète. Maintien d'abduction impossible.
- Faisabilité distance : excellente

---

### Cluster 2 : Coiffe postérieure (infraspinatus / teres minor)

**External Rotation Lag Sign (ERLS)**
- Se 56-98% / Sp 98% / LR+ 28-42
- Meilleur test pour rupture infra/teres. Nécessite un proche pour positionner.
- Faisabilité distance : moyenne (nécessite aide)

**Hornblower's Sign**
- Se 55% / Sp 96%
- Patient tente de porter la main à la bouche, coude en abduction à 90°
- Faisabilité distance : excellente (observable visuellement)

---

### Cluster 3 : Subscapulaire

**Bear Hug Test**
- Se 60-79% / Sp 92%
- Résistance contre une surface fixe si solo
- Faisabilité distance : bonne

**Belly Press (Napoleon)**
- Se 42-71% / Sp 96-98%
- Observable visuellement (position du coude). Pas de résistance requise.
- Faisabilité distance : excellente

---

### Cluster 4 : Instabilité gléno-humérale antérieure

**Apprehension Test**
- Se 47-72% / Sp 96-99% / LR+ 16
- Gold standard instabilité antérieure. Nécessite un proche ou adaptation.
- Faisabilité distance : faible si solo

**Surprise Test (Release Test)**
- Se 64% / Sp 99% / LR+ 64
- Meilleure spécificité. Nécessite un aide impérativement.
- Faisabilité distance : faible

---

### Cluster 5 : Articulation acromio-claviculaire

**Cross Body Adduction**
- Se 77% / Sp 79%
- Auto-faisable : bras tendu en adduction horizontale
- Faisabilité distance : excellente

---

### Note SLAP

Aucun test clinique ne permet d'affirmer ou d'exclure une lésion SLAP de façon fiable (méta-analyse Hegedus). L'O'Brien a une bonne précision pour l'AC joint, pas pour le SLAP. Imagerie confirmative nécessaire si suspicion clinique.

---

### Arbre décisionnel sentinelles

```
Étape 1 — Screening global (toujours)
  Painful arc + Hawkins-Kennedy + Drop Arm

Étape 2 — Ciblage selon résultats
  Coiffe antérieure  → Empty Can + Bear Hug / Belly Press
  Coiffe postérieure → ERLS + Hornblower's
  AC joint           → Cross-body adduction

Étape 3 — Instabilité (contexte traumatique / jeune)
  Apprehension ± aide
```

### Module A — Mobilité gléno-humérale

| Test | Protocole distance | Observable caméra |

## Partie 3 — Scoring et valeurs normatives

### Logique : score 0-2 par test, total sur 18 points

| Score | Couleur | Signification |
|-------|---------|---------------|
| 0 | Vert | Normal |
| 1 | Orange | Borderline / à surveiller |
| 2 | Rouge | Restriction significative |

---

### Module A — Gléno-huméral (/10)

**A1 — Flexion antérieure**
- 0 : ≥ 160° (norme réelle cohorte BMC 2020 : 161° H / 159° F)
- 1 : 140-159°
- 2 : < 140°
- MDC : 12-15°

**A2 — Abduction**
- 0 : ≥ 150° (norme : 152° H / 150° F)
- 1 : 120-149°
- 2 : < 120°

**A3 — Rotation externe à 0°**
- 0 : ≥ 55° (norme : 56° H / 58° F)
- 1 : 40-54°
- 2 : < 40°
- Asymétrie > 10° = indicateur clinique indépendant

**A4 — Rotation interne (main dans le dos)**
- 0 : ≥ T9
- 1 : T10 – L1
- 2 : < L1 (ne dépasse pas la crête iliaque)

**A5 — GIRD (asymétrie RI à 90° abduction)**
- 0 : asymétrie < 10° vs côté sain
- 1 : 10-19°
- 2 : ≥ 20° (seuil validé dans la littérature)
- MDC : 9.4° entre sessions

---

### Module B — Scapulo-thoracique (/8)

**B1 — Apley Scratch (écart bilatéral)**
- 0 : < 5 cm entre les deux mains
- 1 : 5-10 cm
- 2 : > 10 cm ou impossibilité de s'approcher

**B2 — Rotation thoracique quadrupède verrouillée**
- 0 : ≥ 50° bilatéral (seuil « unrestricted » Whetstone/IJSPT 2024)
- 1 : 40-49°
- 2 : < 40°
- MDC : 6° entre sessions (estimation visuelle) / 3-4° avec inclinomètre smartphone

**B3 — Extension thoracique debout**
- 0 : > 30°
- 1 : 20-30°
- 2 : < 20°
- Mesure recommandée : smartphone posé sur le sternum comme inclinomètre (ICC > 0.80)

**B4 — Dyskinésie scapulaire (SDT)**
- 0 : symétrique, fluide, pas de winging
- 1 : légère asymétrie ou élévation prématurée d'un côté
- 2 : winging visible OU asymétrie marquée OU mouvement saccadé
- Limite : ICC 0.60-0.73, qualité vidéo influente — utile surtout pour le suivi longitudinal

---

### Interprétation du score total (/18)

| Score total | Interprétation |
|-------------|----------------|
| 0-3 | Mobilité normale ou très légèrement réduite |
| 4-7 | Restriction modérée — suivi recommandé |
| 8-12 | Restriction significative — bilan présentiel conseillé |
| > 12 | Restriction sévère — orientation clinique |

## Notes d'implémentation MoveCheck

- **Mesure solo :** A1, A2, A3, A4, B1, B2 faisables seul. A5 et B4 nécessitent un proche.
- **Smartphone inclinomètre :** intégrer calibration dans l'app pour A1, A2, A3, B2, B3. Gain précision + données quantifiées entre sessions.
- **MDC pour suivi :** ne conclure à une amélioration réelle que si changement > MDC du test.
- **Ajustement âge :** flexion/abduction diminuent ~15° entre < 30 ans et > 60 ans. Prévoir tranches 20-40 / 40-60 / > 60 à terme.
